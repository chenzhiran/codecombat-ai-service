# 家长用户故事

## 故事 A：孩子从拖拽编程过渡到真实代码

一些孩子已经接触过 Scratch 或图形化编程，家长希望进一步学习真实代码。CodeCombat 通过游戏场景降低真实代码入门难度，让孩子从拖拽过渡到 Python / JavaScript。

**可用于售前表达**：

> 如果孩子已经接触过拖拽式编程，CodeCombat 可以作为下一步：开始写真实代码，但仍然保留游戏化反馈。

## 故事 B：家长关注学习兴趣

家长常担心孩子觉得编程枯燥。CodeCombat 的闯关方式可以先让孩子体验，再决定是否长期学习。

## 使用注意

- 不夸大竞赛、升学、成绩效果。
- 不贬低其他编程启蒙产品。
- 对家长使用稍正式、共情、方案导向的语气。
