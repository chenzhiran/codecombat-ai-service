# 转行者用户故事

## 故事 A：从兴趣到职业准备

一位想转行的用户先用 CodeCombat 建立 Python 基础，再继续学习 Web 开发和项目实践。CodeCombat 解决的是从 0 到 1 的入门阶段，让用户不再害怕写代码。

**可用于售前表达**：

> 如果目标是转行，CodeCombat 更像第一阶段的地基：先把语法、逻辑和编程思维练起来，后面再进入具体方向和项目。

## 故事 B：非科班建立信心

非科班用户常见问题不是“完全学不会”，而是刚开始没有反馈、容易放弃。闯关机制能让用户看到代码即时产生效果，提升继续学习的动力。

## 使用注意

- 不承诺“学完就能就业”。
- 可以说“帮助打基础”，不能说“保证转行成功”。
