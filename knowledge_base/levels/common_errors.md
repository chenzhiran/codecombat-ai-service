# 常见代码错误库

## Python

| 错误 | 用户表现 | 解释 | 建议 |
|---|---|---|---|
| IndentationError | 缩进报错 | Python 用缩进表示代码块 | 统一使用 4 个空格 |
| NameError | 变量未定义 | 名字拼错或大小写错误 | 检查变量名是否一致 |
| SyntaxError | 语法错误 | 括号、冒号、引号缺失 | 检查报错行附近 |
| Infinite Loop | 页面卡住 | 循环没有有效动作或退出条件 | 简化循环，刷新重试 |
| Attack None | 英雄攻击空气 | 未判断敌人是否存在 | 攻击前使用 `if enemy:` |

## JavaScript

| 错误 | 用户表现 | 解释 | 建议 |
|---|---|---|---|
| SyntaxError | 运行失败 | 括号、分号、引号问题 | 检查 `{}`、`()` 是否成对 |
| TypeError | 方法不存在 | 对空对象调用方法 | 调用前判断对象存在 |
| ReferenceError | 变量未定义 | 变量名拼写或作用域错误 | 检查声明位置和拼写 |

## 教学回应原则

- 不只翻译报错，要解释“为什么”。
- 给最小修改建议，不一次性重写全部代码。
- 鼓励用户运行并观察结果。
