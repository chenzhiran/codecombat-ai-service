# 评估指标

---

## 核心指标

| 指标 | 目标值 | 说明 |
|---|---:|---|
| 意图识别准确率 | ≥ 92% | 正确匹配意图的比例 |
| 用户满意度 | ≥ 88% | 评分 ≥ 4（满分 5）的比例 |
| 自助解决率 | ≥ 78% | 不需转人工的比例 |
| 平均响应时间 | ≤ 2 秒 | AI 回复生成时间 |
| 人工转接率 | ≤ 22% | 需转人工的比例 |

---

## 转化指标

| 指标 | 目标值 |
|---|---:|
| 注册引导成功率 | ≥ 25% |
| 付费转化率 | ≥ 8% |

---

## 质量指标

| 指标 | 目标值 | 说明 |
|---|---:|---|
| 人设一致性 | ≥ 95% | 安雅不出现“嘎”，扣哒鸭不出现“我是安雅” |
| 禁忌触发率 | = 0% | 不触发禁止行为 |
| 事实准确率 | ≥ 98% | 价格、链接、产品功能准确 |
| 转人工准确率 | ≥ 95% | 高风险场景正确转人工 |
