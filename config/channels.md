# 渠道配置

## 官网在线客服

| 配置项 | 值 |
|---|---|
| 名称 | 官网在线客服 |
| URL | codecombat.cn |
| 默认人设 | 安雅 |
| 默认 Agent | PresaleAgent |

**功能**：

- 实时对话。
- 页面上下文感知。
- 关卡内问问AI联动。
- 售前和售后分流。

**欢迎消息**：

> 你好！我是安雅 😊  
> 在游戏里我是你的第一个英雄伙伴，在这里我帮你解决问题。  
>
> 请问有什么可以帮你的？  
> 关卡攻略 | 课程咨询 | 会员购买 | 使用问题

**服务时间**：

| 类型 | 时间 |
|---|---|
| 人工客服 | 周一至周五 9:00-18:00 |
| AI 客服 | 24/7 |

## 关卡内问问AI

| 配置项 | 值 |
|---|---|
| 名称 | 问问AI |
| 入口 | 关卡页面左下角 AI 图标 |
| 默认人设 | 问问AI |
| 默认 Agent | TutorialAgent |

**功能**：

- 分析当前关卡目标。
- 根据用户代码定位问题。
- 按 clickCount 渐进式给提示。
- 引导用户理解知识点，而不是直接代写。
