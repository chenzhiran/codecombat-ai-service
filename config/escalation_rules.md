# 转人工规则

---

## 立即转人工

**触发条件**：

- 退款请求。
- 严重投诉。
- 账号安全问题。
- 支付异常。
- 用户使用关键词：投诉、退款、举报、人工客服、真人客服。

---

## 收集信息后转人工

| 触发意图 | 需收集的信息 |
|---|---|
| 修改用户名 | 当前用户名、新用户名 |
| 注销账号 | 用户名、确认意愿 |
| 重置关卡 | 用户名、重置范围 |
| 发票开具 | 发票类型、抬头名称、税号、接收邮箱、订单号 |
| 支付后未开通 | 账号、支付时间、订单截图 |

---

## 不应转人工的情况

- 普通产品介绍。
- 价格咨询。
- 注册入口咨询。
- 浏览器和设备建议。
- 一般学习建议。
- 小红书轻量互动。
